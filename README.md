# 🙏 Igreja Comunidade da Graça - App Oficial

App para cultos, devocionais, células, orações e muito mais.

## ✨ Funcionalidades

- 📅 Agenda de Cultos com confirmação de presença
- 📖 Devocional Diário
- 🎬 Vídeos das mensagens
- 👥 Células com integração WhatsApp
- 💰 Dízimos e PIX
- 📢 Avisos da igreja
- 🙏 Pedidos de Oração
- 📍 Localização e rotas

## 🔧 Acesso Admin

Clique **7 vezes** no cabeçalho da igreja e digite a senha: `admin123`

## 🚀 Deploy

Este projeto está pronto para deploy na Vercel.

## 📝 Personalização

Altere a senha do admin no arquivo `admin.js` na função `loginAdmin()`
